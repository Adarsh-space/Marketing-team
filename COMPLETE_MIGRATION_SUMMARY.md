## üéØ Setup Instructions

### Step 1: Register Zoho App

1. Go to https://api-console.zoho.com/
2. Click **GET STARTED** or **ADD CLIENT**
3. Choose **Server-based Applications**
4. Fill in:
   - Client Name: Your App Name
   - Homepage URL: http://localhost:3000
   - Authorized Redirect URI: http://localhost:3000/zoho/callback
5. Click **CREATE**
6. Copy **Client ID** and **Client Secret**

### Step 2: Register Facebook App

1. Go to https://developers.facebook.com/apps/
2. Create App ‚Üí **Business**
3. Add **Facebook Login** product
4. Configure OAuth:
   - Valid OAuth Redirect URIs: http://localhost:3000/social-media/facebook/callback
5. Add Permissions:
   - pages_manage_posts
   - pages_read_engagement
   - pages_manage_metadata
6. Copy **App ID** and **App Secret**

### Step 3: Configure Environment

Add all credentials to `backend/.env` (see template above)

### Step 4: Start Services

```bash
# Backend
cd backend
pip install -r requirements.txt
uvicorn server:app --reload --port 8000

# Frontend
cd frontend
npm install
npm start
```

### Step 5: Connect Accounts

1. Go to http://localhost:3000/settings
2. Click **Connect to Zoho** ‚Üí Authorize
3. Click **Connect Facebook** ‚Üí Authorize
4. Click **Connect Instagram** ‚Üí Authorize (via Facebook)
5. Ready to use!

## ‚ö†Ô∏è Important Notes

### Social Media Limitations

**Facebook:**
- ‚úÖ Can post to pages (with page admin access)
- ‚úÖ Text, images, links supported
- ‚ùå Cannot post to personal timeline
- ‚ùå Video posting requires upload API

**Instagram:**
- ‚úÖ Can post to Business/Creator accounts only
- ‚úÖ Images and captions
- ‚ùå Requires Facebook page connection
- ‚ùå Image must be publicly accessible URL
- ‚ùå Cannot post stories via API

**Zoho Social:**
- ‚ùå No API available
- ‚ö†Ô∏è Manual posting required

### Token Management

- **Zoho:** Tokens expire after 1 hour (auto-refresh implemented)
- **Facebook:** Tokens expire after 60 days (refresh needed)
- **Instagram:** Uses Facebook tokens

### API Rate Limits

- **Zoho CRM:** 5,000 calls/day
- **Zoho Mail:** Varies by account type
- **Zoho Campaigns:** 100 calls/minute
- **Facebook:** 200 calls/user/hour
- **Instagram:** 200 calls/user/hour
